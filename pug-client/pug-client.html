pug-client.js uses Pug's compileClient module to fuse together multiple Pug templates into a single function that can be used on the client.

pug-client.js must be run in an environment where require('pug') will work.

When pug-client.js is run, it checks to see if a config.json file exists in its directory. If it doesn't, it creates one, as well as creating a direcotry named src.

watcher.js monitors changes in the pug source directory and updates the client function file accordingly.
