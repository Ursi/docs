popup(elem, options[, callback])

  1. elem: HTMLElement

    2. The element that will be popped up

  2. options: Object
    1. reference: HTMLElement

      2. an element used for positioning the popup, by default the viewport is used

    2. xe: Number

      3. a value from 0 to 1, (0 being the rightmost side of the element/viewport and 1 being the leftmost side) that is matched up with the x value specified by xr

    3. xr: Number

      4. a value from 0 to 1, (0 being the rightmost side of the element/viewport and 1 being the leftmost side) that is matched up with the x value specified by xe

    4. ye: Number

      5. a value from 0 to 1, (0 being the topmost side of the element/viewport and 1 being the bottommost side) that is matched up with the y value specified by yr

    5. yr: Number

      6. a value from 0 to 1, (0 being the topmost side of the element/viewport and 1 being the bottommost side) that is matched up with the y value specified by ye

    6. dim: Number

      7. the opacity of the black background from 0â€“255

    7. boxShadow: String

      8. the value of the CSS box-shadow property of the popup

    8. zIndex: Number

      9. the value of the CSS z-index property

  3. callback: Function
    1. elem: HTMLElement

      2. the elem parameter passed into popup

    2. options: Object

      3. the options parameter passed into popup

    4. a callback to be executed when the popup is closed via clicking outside of it
